RootCloak

This Android app is a module for the Xposed Framework. Some apps, such as banking, enterprise, or audio/video apps will not run if your phone is rooted. This allows you to run apps that detect root without disabling root.


Downloads: Xposed Module Repo
Support: XDA Thread
Source Code: GitHub Repo
